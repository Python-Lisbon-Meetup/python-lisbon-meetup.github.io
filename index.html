Python Lisbon Meetup

A welcoming community for Python developers, learners, and enthusiasts in Lisbon.

About the Meetup

The Python Lisbon Meetup brings together people interested in the Python programming language. Whether you are a beginner, an experienced developer, or simply curious about Python, you are welcome.

We focus on sharing knowledge, discussing ideas, and connecting with fellow Pythonistas in a friendly and inclusive environment.

When We Meet

We meet on the first non-holiday Thursday of each month.

Get Involved

  • Join the conversation on our Discord server.
  • Find upcoming events on Meetup.com .

Code of Conduct

The Python Lisbon Meetup is dedicated to providing a safe, welcoming, and inclusive experience for everyone.

Every participant, speaker, organiser, and everyone involved with the Python Lisbon Meetup must follow this Code of Conduct. This applies to all in-person events as well as all online interactions, including Discord and any other communication channels.

We welcome people of all backgrounds and identities. Harassment, discrimination, or exclusionary behaviour is not acceptable. This includes behaviour based on (but not limited to) sociodemographic characteristics, personal background, experience level with Python or programming, or any other personal attribute.

Be respectful, be kind, and assume good intent. If someone asks you to stop a behaviour, you are expected to comply immediately.

Reporting Issues

If you experience or witness a violation of this Code of Conduct, please reach out to one of the Code of Conduct team members listed below. All reports will be taken seriously and handled with care and discretion.
